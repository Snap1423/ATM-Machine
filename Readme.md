# 🏧 ATM Machine in Python
A simple **ATM Machine project** using Python's `match-case` feature (Python 3.10+).  
This project includes:
- ✅ PIN Verification  
- ✅ Balance Check  
- ✅ Deposit Money  
- ✅ Withdraw Money  
- ✅ Exit Option  

---

## 🚀 Run the Program
python atm.py
